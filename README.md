# Moshify

This is static site build based on page moshified.com with pure HTML and CSS + JavaScript for dropdowns and animations.

## Sections of the project

- NavBar
- Hero
- Main Page
- Footer

## Technologies used

- HTML
- CSS
- JavaScript

## Instructions

- Clone or download zipped code
- Run terminal in your IDE
- Run npm install / npm i
- Run parcel index.html
